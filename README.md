（5）打开nemu/scripters/build.mk文件, 你会在CFLAGS变量中看到gcc的一些编译选项. 请解释gcc中的-Wall和-Werror有什么作用? 为什么要使用-Wall和-Werror?
    答：
    -Wall：输出所有错误或警告信息。
    -Werror：将所有的警告当成错误进行处理，并中断编译过程。
    使用-Wall和-Werror是为了尽可能地帮助找到存在的错误。
